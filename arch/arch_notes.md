# `pacman`

| Action                 | Command                |
|------------------------|------------------------|
| Search                 | pacman -Ss package     |
| Install                | pacman -S package      |
| Remove, keep deps      | pacman -R package      |
| Remove, including deps | pacman -Rs package     |
| Rem, inc deps + config | pacman -Rns package    |
| Upgrade packages       | pacman -Syu            |
