Helix

Action Shortcut
Open tutorial hx --tutor
Check LSP status hx --health, or hx --health language
Open 1 file, >1 file, or a direcory hx file.rs, hx f1.rs f2.rs, hx .
Open command mode : (colon), use tab and shift+tab
Quit, Write+Quit, Write+Quit All :q or :quit, :wq, :wqa
Force quit (don't save) :q!
Open file :o file.rs, also :e or :edit
Print the current working directory :pwd
Change directory :cd dir
Change theme temporarily :theme, tab between themes
Open the helix config :open-config

Every movement in Helix is a selection.

Action Vim-Style Keybinding
h j k l left up down right
e Jump to end of next word
w Jump to beginning of next word
b Jump to end of previous word
W E B See prev, ignore punctuation
f +, e.g. f + , (find) Jump to next occurance of symbol
t + (until) Same as f, but stops short of symbol
F T See f, t, but goes backwards
ctrl + u, ctrl + d Move half a page up or down
ctrl + f, ctrl + b Move full page forwards or backwards
G, e.g. 15G Go to line, e.g. line 15
command Repeat command. E.g. 3w moves 3 words
x Select whole line
Press x multiple times Select multiple lines
; or just move Collapse selection (deselect)
